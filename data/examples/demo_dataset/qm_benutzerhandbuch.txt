ProQuality Manager v4.2

WILLKOMMEN BEI PROQUALITY MANAGER

ProQuality Manager ist Ihr umfassendes Qualitätsmanagementsystem für die Erfüllung von ISO 9001:2015, ISO 14001:2015 und IATF 16949:2016 Standards. Dieses Handbuch führt Sie durch alle Funktionen und Arbeitsabläufe.

ERSTE SCHRITTE

1. SYSTEMZUGANG
- Öffnen Sie Ihren Webbrowser
- Navigieren Sie zu https://qm.ihreunternehmen.de
- Geben Sie Ihre Anmeldedaten ein
- Klicken Sie auf "Anmelden"

2. DASHBOARD-ÜBERSICHT
Nach der Anmeldung sehen Sie das Hauptdashboard mit:
- Aktuelle Kennzahlen
- Offene Aufgaben
- Terminübersicht
- Systembenachrichtigungen

3. NAVIGATION
Die Hauptnavigation befindet sich in der linken Seitenleiste:
- Dokumente: Dokumentenmanagement
- Audits: Audit-Verwaltung
- CAPA: Korrekturmaßnahmen
- Risiken: Risikomanagement
- Lieferanten: Lieferantenbewertung
- Berichte: Reporting und Analytik

DOKUMENTENMANAGEMENT

1. DOKUMENTE ANZEIGEN
- Klicken Sie auf "Dokumente" in der Navigation
- Verwenden Sie Filter nach Typ, Status oder Abteilung
- Suchfunktion für schnelles Auffinden
- Dokumentenvorschau durch Klick auf Titel

2. NEUES DOKUMENT ERSTELLEN
- Klicken Sie auf "Neues Dokument"
- Wählen Sie Dokumententyp aus:
  * Qualitätshandbuch (QHB)
  * Verfahrensanweisung (VA)
  * Arbeitsanweisung (AA)
  * Prüfanweisung (PA)
  * Formular/Checkliste
- Füllen Sie die Metadaten aus:
  * Titel und Beschreibung
  * Verantwortliche Person
  * Gültigkeitsdauer
  * Verteilerliste
- Laden Sie das Dokument hoch oder erstellen Sie es im Editor

3. DOKUMENTENBEARBEITUNG
- Klicken Sie auf "Bearbeiten" bei einem Dokument
- Verwenden Sie den integrierten Editor oder
- Laden Sie eine neue Version hoch
- Fügen Sie Änderungskommentare hinzu
- Senden Sie zur Genehmigung

4. GENEHMIGUNGSWORKFLOW
- Dokumente durchlaufen automatisch den Genehmigungsprozess
- Genehmiger erhalten E-Mail-Benachrichtigungen
- Status-Tracking: Entwurf → Prüfung → Genehmigt → Veröffentlicht
- Kommentare und Änderungsanfragen möglich

5. VERSIONSKONTROLLE
- Automatische Versionierung bei jeder Änderung
- Vergleich zwischen Versionen
- Rollback zu früheren Versionen möglich
- Änderungshistorie mit Zeitstempel

AUDIT-MANAGEMENT

1. AUDIT-PLANUNG
- Navigieren Sie zu "Audits" → "Planung"
- Erstellen Sie einen Jahresauditplan
- Terminierung und Auditorenzuweisung
- Automatische Kalendererinnerungen

2. AUDIT-CHECKLISTEN
- Wählen Sie passende Checklisten aus der Bibliothek
- Anpassung an spezifische Anforderungen
- Bewertungsskalen konfigurieren
- Mobile Nutzung auf Tablets/Smartphones

3. AUDIT-DURCHFÜHRUNG
- Starten Sie das Audit über "Audits" → "Durchführung"
- Arbeiten Sie die Checkliste systematisch ab
- Dokumentieren Sie Befunde mit Fotos
- Bewerten Sie Konformität (konform/Abweichung/Beobachtung)
- Erfassen Sie Maßnahmen direkt im System

4. AUDIT-BERICHTE
- Automatische Berichterstellung nach Audit-Abschluss
- Standardisierte Berichtsvorlagen
- Individuelle Anpassungen möglich
- E-Mail-Versand an Beteiligte

KORREKTURMASSNAHMEN (CAPA)

1. MÄNGEL ERFASSEN
- Navigieren Sie zu "CAPA" → "Neuer Mangel"
- Kategorisierung nach Quelle:
  * Kundenreklamation
  * Interne Abweichung
  * Audit-Befund
  * Lieferantenprobleme
- Priorität festlegen (niedrig/mittel/hoch/kritisch)
- Verantwortlichkeiten zuweisen

2. ROOT-CAUSE-ANALYSE
- Verwenden Sie integrierte Analyse-Tools:
  * 5-Why-Methode
  * Ishikawa-Diagramm
  * Pareto-Analyse
- Dokumentieren Sie Ursachen systematisch
- Verknüpfen Sie mit ähnlichen Fällen

3. MASSNAHMENPLANUNG
- Definieren Sie Sofortmaßnahmen
- Planen Sie Korrekturmaßnahmen
- Legen Sie Präventivmaßnahmen fest
- Terminierung und Budgetplanung

4. UMSETZUNGSVERFOLGUNG
- Automatische Erinnerungen an Verantwortliche
- Status-Updates und Fortschrittsberichte
- Dokumentation der Umsetzung
- Foto-Dokumentation von Verbesserungen

5. WIRKSAMKEITSPRÜFUNG
- Automatische Erinnerung zur Wirksamkeitsprüfung
- Bewertung der Zielerreichung
- Erfolgsmessung anhand von Kennzahlen
- Abschlussfreigabe durch Qualitätsmanager

RISIKOMANAGEMENT

1. RISIKEN IDENTIFIZIEREN
- Navigieren Sie zu "Risiken" → "Neues Risiko"
- Kategorisierung nach Bereichen:
  * Produktrisiken
  * Prozessrisiken
  * Umweltrisiken
  * Compliance-Risiken
- Beschreibung und Kontext dokumentieren

2. RISIKOBEWERTUNG
- Bewertung der Eintrittswahrscheinlichkeit (1-5)
- Bewertung der Auswirkungen (1-5)
- Automatische Risikoscore-Berechnung
- Visualisierung in Risikomatrix

3. RISIKOMINDERUNG
- Maßnahmenplanung zur Risikominderung
- Verantwortlichkeiten und Termine
- Budgetplanung für Maßnahmen
- Umsetzungsüberwachung

4. RISIKO-MONITORING
- Regelmäßige Neubewertung
- Trend-Überwachung
- Frühwarnsystem bei kritischen Risiken
- Eskalationsverfahren

- Erfolgskontrollen

HÄUFIGE FRAGEN (FAQ)

F: Wie kann ich mein Passwort zurücksetzen?
A: Klicken Sie auf "Passwort vergessen" auf der Anmeldeseite und folgen Sie den Anweisungen.

F: Warum kann ich ein Dokument nicht bearbeiten?
A: Prüfen Sie Ihre Berechtigungen oder ob das Dokument von einem anderen Benutzer bearbeitet wird.

F: Wie kann ich Berichte automatisch versenden?
A: Nutzen Sie die Funktion "Berichts-Abonnements" in den Berichtseinstellungen.

F: Kann ich das System offline nutzen?
A: Die Mobile App bietet Offline-Funktionen für Audits. Synchronisation erfolgt bei Internetverbindung.

F: Wie importiere ich Daten aus dem alten System?
A: Kontaktieren Sie den Support für Datenimport-Services.
