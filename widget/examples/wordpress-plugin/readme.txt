Add an AI-powered chat widget to your WordPress site using RAG (Retrieval-Augmented Generation) technology.

== Description ==

The RAG Chat Widget plugin allows you to easily add an intelligent chat assistant to your WordPress website. The widget uses RAG (Retrieval-Augmented Generation) technology to provide accurate, context-aware responses based on your uploaded documents.

**Key Features:**

* **Easy Integration**: Add the widget to your site with just a few clicks
* **Customizable Appearance**: Choose from multiple themes and customize colors
* **Flexible Positioning**: Place the widget in any corner of your site
* **Mobile Responsive**: Works perfectly on all devices
* **Page-Specific Control**: Choose which pages to display the widget on
* **Real-time Chat**: Instant responses with typing indicators
* **Document-Based Responses**: Answers are based on your uploaded documents
* **Secure API Integration**: Uses API keys for secure communication

**How It Works:**

1. Install and activate the plugin
2. Configure your RAG API server details
3. Upload documents to your RAG system
4. The widget will provide intelligent responses based on those documents

**Perfect For:**

* Customer support and FAQ
* Documentation assistance
* Knowledge base integration
* Educational content
* Research and information retrieval

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rag-widget` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to Settings > RAG Widget to configure the plugin.
4. Enter your RAG API key and server URL.
5. Customize the appearance and behavior as needed.
6. Test the connection to ensure everything is working properly.

== Frequently Asked Questions ==

= What is RAG? =

RAG stands for Retrieval-Augmented Generation. It's an AI technique that combines information retrieval with text generation to provide accurate, source-backed responses.

= Do I need my own RAG server? =

Yes, you need to have a RAG API server running. This plugin connects to your RAG server to provide chat functionality.

= How do I get an API key? =

The API key is generated by your RAG server. Refer to your RAG server documentation for instructions on generating API keys.

= Can I customize the widget appearance? =

Yes! You can customize colors, themes, position, messages, and more through the plugin settings.

= Is the widget mobile-friendly? =

Absolutely! The widget is fully responsive and works great on all devices.

= Can I control which pages show the widget? =

Yes, you can choose to show the widget on all pages, specific page types, or configure custom rules.

= How do I add documents to the RAG system? =

Documents are uploaded directly to your RAG server, not through this WordPress plugin. Use your RAG server's document upload functionality.

== Configuration ==

After installation, configure the plugin by going to **Settings > RAG Widget** in your WordPress admin.

**Required Settings:**
* **API Key**: Your RAG server API key
* **API URL**: The URL of your RAG server (e.g., https://your-rag-server.com)

**Optional Settings:**
* **Theme**: Choose from default, dark, blue, green, purple, orange, or red
* **Position**: Bottom-right, bottom-left, top-right, or top-left
* **Colors**: Customize primary and secondary colors
* **Text**: Customize title, welcome message, and placeholder text
* **Pages**: Choose which pages to display the widget on

**Testing:**
Use the "Test Connection" button to verify that your WordPress site can communicate with your RAG server.
